# JavaFX 3D Objektum Viewer

Egyszeru Java projekt, ami egy 3D hengert es egy kockat jelenit meg egymason.
Az objektum egerhuzassal forgathato, igy minden oldalrol megnezheto.

## Kovetelmenyek

- Java 17 vagy ujabb
- Maven 3.9+

## Build

```bash
mvn clean package
```

## Futtatas

```bash
mvn javafx:run
```
